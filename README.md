# 📊 Exploratory Data Analysis Project

## 📌 Overview

This project performs **Exploratory Data Analysis (EDA)** using Python to clean, analyze, and visualize the dataset. The objective is to identify patterns, trends, and insights that can support data-driven decision-making.

---

## 🎯 Objectives

- Understand the dataset structure
- Perform data cleaning and preprocessing
- Handle missing values and duplicates
- Explore relationships between variables
- Visualize important trends and patterns
- Generate meaningful business insights

## 🚀 How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/todakarsamruddi-cpu/WHO-Analysis-.git
```

### 2. Navigate to the Project Folder

```bash
cd exploratory-data-analysis
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Launch Jupyter Notebook

```bash
jupyter notebook
```

### 5. Open the Notebook

```
exploratory-data-analysis-project.ipynb
```

Run all cells to reproduce the analysis.
